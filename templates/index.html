Sobre el Proyecto

¿Qué es este proyecto?

Este es un predictor de porcentaje de grasa corporal desarrollado con Machine Learning, específicamente usando el algoritmo XGBoost. El modelo fue entrenado con más de 20,000 registros de datos de lifestyle que incluyen información sobre ejercicio, nutrición y características físicas.

Tecnologías Utilizadas

  • Python 3.11 - Lenguaje de programación
  • Flask - Framework web
  • XGBoost - Algoritmo de ML
  • Scikit-learn - Preprocesamiento y métricas
  • Bootstrap 5 - Framework CSS
  • Render - Hosting y deployment

Métricas del Modelo

  • R² Score: > 0.85 (El modelo explica más del 85% de la varianza)
  • MAE: ~2-3% (Error promedio de predicción)
  • Features: 54+ variables analizadas
  • Validación: Cross-validation K-Fold

Dataset

Fuente: Kaggle - Life Style Data

El dataset incluye información sobre hábitos de vida, ejercicio, nutrición y características físicas de más de 20,000 individuos.

Disclaimer

Importante: Este predictor es una herramienta educativa y no debe ser utilizado como sustituto de evaluación médica profesional. Para obtener mediciones precisas de composición corporal, consulta con un profesional de la salud.
